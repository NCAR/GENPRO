NCAR/EOL GENPRO-I Conversion Utility & Documentation

View on GitHub

Getting the source

$ git clone https://github.com/ncareol/GENPRO.git

Compiling

genpro2nc has no external dependencies.

$ cd GENPRO/g1nc/c
$ make

Documentation

A PDF document describing the GENPRO-I format is available: genpro.pdf
